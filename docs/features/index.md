# Features

Volcano is packed with features to help you create beautiful documentation sites.

## Content

- **[[markdown-syntax]]** — Full GitHub Flavored Markdown support
- **[[wiki-links]]** — Obsidian-style `[[Page Name]]` links
- **[[admonitions]]** — Note, tip, warning, and info callouts
- **[[code-blocks]]** — Syntax highlighting with copy button

## Navigation

- **[[navigation]]** — Sidebar tree, breadcrumbs, TOC, and more

## Customization

- **[[theming]]** — Built-in themes and custom CSS support

## Automation

- **[[auto-index]]** — Automatic folder index pages
- **[[reading-time]]** — Estimated reading time display
- **[[seo-and-meta]]** — Auto-generated SEO tags

## Quality

- **[[reference/cli#link-validation|Link validation]]** — Automatic broken link detection during build
