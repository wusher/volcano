# AI Prompts

Ready-to-use prompts for generating content and customizing your Volcano site with AI assistants like Claude or ChatGPT.

## Available Prompts

### [[custom-theme|Generate a Custom Theme]]
Create a custom CSS theme tailored to your site's personality and brand. Includes examples for minimalist, tech docs, blog, and knowledge base themes.

**Use when**: You want a unique look beyond the built-in themes

### [[generate-docs|Generate Documentation]]
Generate comprehensive documentation for your project by analyzing your source code, including guides, examples, and API reference.

**Use when**: Starting a new documentation site or generating API docs from code

### [[add-github-pages|Deploy to GitHub Pages]]
Set up automated deployment to GitHub Pages with GitHub Actions.

**Use when**: Hosting your site on GitHub

## How to Use These Prompts

1. **Copy the prompt** from any page above
2. **Fill in your specific details** in the bracketed sections
3. **Paste into your AI assistant** (Claude, ChatGPT, etc.)
4. **Review and customize** the generated output
5. **Iterate** if needed with follow-up requests

## Tips for Best Results

- **Be specific**: The more details you provide, the better the output
- **Iterate**: Start with a basic prompt, then refine
- **Context matters**: Mention your target audience and use case
- **Ask for variations**: Request multiple options to choose from
- **Combine prompts**: Use multiple prompts for different aspects

## Example Workflow

1. Use **Generate Documentation** to create your docs from source code
2. Use **Generate a Custom Theme** to style your site
3. Use **Deploy to GitHub Pages** to publish it
