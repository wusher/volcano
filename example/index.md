# Welcome to Volcano

Volcano is a fast and simple static site generator that converts markdown files into a beautiful, navigable website.

## Features

- **Fast Generation** - Build your entire site in seconds
- **Tree Navigation** - Automatic sidebar navigation from folder structure
- **Dark Mode** - Built-in light/dark mode with system preference detection
- **Mobile Friendly** - Responsive design with drawer navigation
- **Clean URLs** - Pretty URLs without `.html` extensions
- **Syntax Highlighting** - Code blocks with beautiful syntax colors

## Quick Start

```bash
# Generate a static site
volcano ./docs -o ./public --title="My Docs"

# Serve the generated site
volcano -s -p 8080 ./public
```
