# Template Examples

Here are practical examples for common patterns in Medusa templates.

## Blog Post Loop

Display all posts with dates and excerpts:

```jinja
{% for post in pages.group("posts").sorted() %}
<article class="mb-8">
  <h2>
    <a href="{{ post.url }}">{{ post.title }}</a>
  </h2>
  <time class="text-sm text-gray-500">
    {{ post.date.strftime('%B %d, %Y') }}
  </time>
  <p>{{ post.description }}</p>
</article>
{% endfor %}
```

## Tag List

Show all tags with post counts:

```jinja
<ul class="flex gap-2">
  {% for tag, tag_pages in tags.items() %}
  <li>
    <span class="bg-gray-100 px-2 py-1 rounded">
      {{ tag }} ({{ tag_pages | length }})
    </span>
  </li>
  {% endfor %}
</ul>
```

## Recent Posts Sidebar

Display the 5 latest posts:

```jinja
<aside>
  <h3>Recent Posts</h3>
  <ul>
    {% for post in pages.group("posts").latest(5) %}
    <li>
      <a href="{{ post.url }}">{{ post.title }}</a>
    </li>
    {% endfor %}
  </ul>
</aside>
```

## Reusable Card Partial

Create a card component in `site/_partials/card.html.jinja`:

```jinja
<div class="bg-white shadow rounded-lg p-4">
  <h3 class="font-bold">{{ title }}</h3>
  <p class="text-gray-600">{{ description }}</p>
  {% if link %}
  <a href="{{ link }}" class="text-blue-500">Read more</a>
  {% endif %}
</div>
```

Use it in your templates:

```jinja
{% include "card.html.jinja" %}
```

## Navigation from Data

Define navigation in `data/nav.yaml`:

```yaml
- label: Home
  url: /
- label: Blog
  url: /posts/
- label: About
  url: /about/
```

Render it in your layout:

```jinja
<nav>
  {% for item in data.nav %}
  <a href="{{ item.url }}"
     class="{% if current_page.url == item.url %}font-bold{% endif %}">
    {{ item.label }}
  </a>
  {% endfor %}
</nav>
```

## Social Links

Define social links in `data/social.yaml`:

```yaml
- platform: GitHub
  url: https://github.com/username
- platform: Twitter
  url: https://twitter.com/username
```

Render them:

```jinja
<footer>
  {% for link in data.social %}
  <a href="{{ link.url }}">{{ link.platform }}</a>
  {% endfor %}
</footer>
```

## Table of Contents

Add a table of contents to long posts:

```jinja
{% if current_page.toc | length > 2 %}
<nav class="toc mb-8 p-4 bg-gray-50 rounded">
  <h4 class="font-bold mb-2">Contents</h4>
  {{ render_toc(current_page) }}
</nav>
{% endif %}

{{ current_page.content | safe }}
```

## Conditional Content

Show different content for drafts:

```jinja
{% if current_page.draft %}
<div class="bg-yellow-100 p-4 mb-4">
  This is a draft and won't appear in production.
</div>
{% endif %}
```
